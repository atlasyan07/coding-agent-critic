# Critic-Guided Local Coding Agent: Offline RL for Trajectory-Level Control

This repository hosts the public research note for an active work-in-progress project on building a capable coding agent for resource-constrained environments. The system uses an actor-controller-critic architecture, where a small fine-tuned model is made more reliable through system structure, constrained actions, and offline learning rather than raw model scale alone.

This is an active research project. The codebase is currently private and not yet released.
